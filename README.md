* 常用操作
> 进入容器
>* GO： `docker exec -it go/ bin/bash`      
>* RSS：`docker exec -it rss/ bin/bash`      
>* PIC：`docker exec -it pic/ bin/bash`      

> 启动      
>* GO： `./go-cqhttp/go-cqhttp faststart`     
>* RSS：`cd cq-picsearcher-bot`  `npm run test`     
>* PIC：`cd ELF_RSS-x.x.x`       `nb run`
